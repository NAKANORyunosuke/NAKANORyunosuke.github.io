# Debian13 gnome環境でC#の開発をしてみる
## .NET SDKを入れる
Microsoftのapt署名鍵をインストールとaptリポジトリの追加
```
curl -fsSL https://packages.microsoft.com/keys/microsoft-2025.asc | sudo gpg --dearmor -o /usr/share/keyrings/microsoft-prod.gpg
wget https://packages.microsoft.com/config/debian/13/prod.list -O /etc/apt/sources.list.d/microsoft-prod.list
```

```
sudo apt update -y \
sudo apt install -y dotnet-sdk-10.0
```
